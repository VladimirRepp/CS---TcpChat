# CS---TcpChat
Examples of chat with server and client app on tcp socket

Using:
- C#
- System.Net.Sockets
- .Net9
- Console (for client and server)
- IDE: MVS 2022

For launch:
- First run TcpServer.exe
- Second run several TcpClientApp.exe
